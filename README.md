## ✨ Features

- 🎯 **Typing Animation** — Dynamic role titles with smooth typewriter effect
- 🔍 **Scroll Reveal** — Sections animate into view using Intersection Observer
- 🧑‍💻 **About Me** — Education, focus areas, goals, and quick info card
- 🛠️ **Skills Showcase** — Categorized skill cards (Frontend, Backend, Database, Tools)
- 📂 **Projects Section** — Highlighted projects with tech stack and GitHub links
- 📧 **Contact** — Direct email link for quick communication
- 📄 **Resume Download** — One-click downloadable resume
- 📱 **Fully Responsive** — Optimized for all screen sizes

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/Madhan-29/Madhan_Portfolio.git

# Navigate to project directory
cd Madhan_Portfolio/madhan-portfolio

# Install dependencies
npm install

# Start the development server
npm start
```

The app will open at **http://localhost:3000** 🎉

### Build for Production

```bash
npm run build
```
